Задание №5

Прочитать про единицы измерения цветов. Какими способами можно обозначить цвет? запишите ответ списком на вашем сайте
Ответ:
В CSS существует несколько вариатов указать цвет, а если быть точнее 3.

  • Первый, самый простой это с помощью указания цвета. Но есть ограничение в 145 названий цветов, но и их еще нужно все запомнить.

  • Второй- это с помощью указания RGB кода цвета например (192,192,192)

  • Третий- это с помощью HEX кода например #000000.
